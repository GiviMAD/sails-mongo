# sails-mongo
[![npm version](https://badge.fury.io/js/sails-mongo.svg)](http://badge.fury.io/js/sails-mongo)

Sails.js/Waterline adapter for MongoDB.

## Installation

Install from NPM.

```bash
$ npm install sails-mongo
```
